# 更新日志 (CHANGELOG)

## [v1.3.0] - 2026-05-12
### ✨ 新增功能
- **无限滚动 (瀑布流)**：移动端和 iPad 端新增无限滚动功能，下拉到底部自动加载下一页，模拟原生 App 体验。同时在移动端隐藏了冗余的分页组件。
- **响应式交互优化**：PC 端保留标准分页器，移动端切换为平滑的自动加载模式，性能更佳。
- **加载状态反馈**：移动端列表底部新增“加载中”、“没有更多了”等视觉提示。

## [v1.2.0] - 2026-05-12
### ✨ 新增功能
- **双布局模式**：新增针对不同设备的自适应布局。PC 端采用高效表格流，移动端/iPad 采用瀑布流卡片展示（一行一个）。
- **画质自动识别**：根据电影文件名自动标注 4K、1080P、720P 等画质标签，并支持分类筛选。
- **站外搜索集成**：在搜索栏新增“站外搜索”独立按钮，支持一键直达 `buerchen.top` 资源库。
- **UI 视觉升级**：全面优化搜索框阴影、卡片圆角、按钮交互及配色方案，提升视觉高级感。

### ⚙️ 逻辑优化
- **Excel 解析增强**：引入智能 Fallback 机制。支持 C 列（中文名）、D 列（磁力）的硬编码位置读取及全行扫描识别。
- **磁力链接交互**：磁力链接默认缩短展示，支持点击展开/收起，并针对超长链接进行了 CSS 换行优化。
- **中文国际化**：配置 Element Plus 全局中文语言包，分页组件现已完美汉化。

---

## [v1.1.0] - 2026-05-12
### ✨ 新增功能
- **分页加载**：引入分页机制，支持 10/20/50 条数据切换，极大提升了大数据量下的页面性能。
- **复制可靠性**：新增多方案复制逻辑（Navigator + Textarea 兜底），确保磁力链接 100% 成功复制。

### 🔧 部署与修复
- **GitHub Actions 升级**：升级部署脚本至 Node.js 24，解决 GitHub 官方弃用警告。
- **部署路径修复**：修正 `base` 路径配置，解决 GitHub Pages 部署后 404 及资源无法加载的问题。

---

## [v1.0.0] - 2026-05-12
### 🎉 初始发布
- **核心功能**：基于 Vue 3 + Vite + Element Plus 构建。
- **数据驱动**：支持解析 `public/data/movies.xls` 格式文件。
- **模糊搜索**：集成 Fuse.js 算法，实现毫秒级的名称模糊匹配。
- **自动部署**：配置 GitHub Actions，实现推送代码自动发布。
